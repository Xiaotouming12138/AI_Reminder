## EN

### âœ¨ Overview
The AI Reminder Assistant is a lightweight desktop application designed to provide timely, customizable reminders generated by a local or cloud-based LLM. 

### ğŸ’¡ Features
* **AI-Powered Reminders:** Generates unique, personalized reminders based on a user-defined prompt (e.g., "Stay focused," "Drink water," "Take a break").
* **Flexible Scheduling:** Supports two modes:
    * **Daily Time:** Set a specific time (e.g., `14:30`) for a reminder.
    * **Interval:** Set reminders to pop up every N minutes (e.g., `60` minutes).
* **LLM API Compatibility:** Works with Ollama (local models) and standard OpenAI-compatible API endpoints.

### âš™ï¸ Installation and Setup

**Install dependencies:**
    ```bash
    pip install -r requirements.txt
    ```
    *(Note: You need to create `requirements.txt` based on your imported libraries, e.g., `customtkinter`, `requests`, `schedule`, `plyer`, `pystray`, `Pillow`.)*

**Run the application:**
    ```bash
    python main.py
    ```

### ğŸ› ï¸ Configuration (Settings Tab)

1.  **API URL:** Enter the endpoint for your LLM (e.g., `http://localhost:11434/api/generate` for Ollama, or an OpenAI-compatible URL).
2.  **Model:** Select the model available on your API server (e.g., `llama3`, `mistral`).
3.  **Prompt Tab:** Define the system prompt for the AI (e.g., "You are an encouraging assistant. Generate a short, positive work reminder.").
4.  **Schedule Tab:** Choose between **Daily Time** or **Interval (Minutes)** and set the corresponding value.
5.  **Run on Startup:** Enable this option to automatically run the application when your computer boots.
